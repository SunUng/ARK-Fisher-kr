# Installation
> - (1). Install ahk from [(https://autohotkey.com/)]
> - (2). Download Ark-fisherman.ahk
> - (3). Compile Script using CMD, GUI || Right Click file.ahk > Compile Script 

# Usage

- START SCRIPT: F5
- STOP SCRIPT: F6

> - (1). Run Compiled ahk file
> - (2). Enter Game
> - (3). Sit Down
> - (4). Cast Rod, Press F5
> - (5). Wait and enjoy your 2.5's LOOTS
